## Medical Diagnostics API 

Этот проект представляет собой микросервис для диагностики медицинских показателей на базе датасета Breast Cancer. Сервис использует модель RandomForest  для предсказания типа опухоли на основе 5 числовых параметров.

Структура проекта:
* train.py — Загрузка данных, обучения модели и её сохранения.
* main.py — Основной код API на FastAPI.
* model.joblib — Обученая модели.
* requirements.txt — Список зависимостей Python.
* Dockerfile — Инструкции для сборки образа Docker.

---

## Как запустить проект

### 1. Локальный запуск без Docker

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
Запустите сервер:

uvicorn main:app --reload
Сервер будет доступен по адресу: http://127.0.0.1:8000

### 2. Локальный запуск с Docker
сперва нужно запустить Docker

Потом собрать образ:
docker build -t medical-ml-api .

Запустить контейнер:
docker run -p 8000:8000 medical-ml-api

## API
После запуска нужно перейти в браузере по адресу:
http://127.0.0.1:8000/docs

Там вы найдете интерактивную документацию Swagger, где можно нажать кнопку "Try it out" и отправить тестовые данные в формате JSON:

```JSON
{
  "mean_radius": 14.0,
  "mean_texture": 19.0,
  "mean_perimeter": 90.0,
  "mean_area": 600.0,
  "mean_smoothness": 0.09
}
```
